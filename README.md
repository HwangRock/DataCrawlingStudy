# DataCrawlingStudy
AI와데이터분석 팀의 데이터 크롤링 스터디를 기록한 레포지토리입니다.



## 데이터 크롤링이 나에게 필요한 기술일까?
1. NLP 모델을 활용한 프로젝트에서 모델의 학습 데이터 수집이나 실시간 데이터 수집을 통해서 더 좋은 성능의 모델을 개발할 수 있기에 필요하다.
2. 크롤링 데이터를 통해서 데이터 기반의 API를 구현해서 좋은 서비스를 개발하는 엔지니어로 성장할 수 있다.
